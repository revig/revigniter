Getting Started With revIgniter

Any software application requires some effort to learn. revIgniter aims to minimize the learning curve while making the process as enjoyable as possible.

The first step is to install revIgniter, then read all the topics in the Introduction section of the Table of Contents.

Next, read each of the General Topics pages in order. Each topic builds on the previous one, and includes code examples that you are encouraged to try.

Once you understand the basics you'll be ready to explore the Library Reference and Helper Reference pages to learn to utilize the native libraries and helper files.
